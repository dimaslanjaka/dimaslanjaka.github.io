SOME TYPES OF BORDER CSS

Created2017-09-28|Updated2017-09-28|uncategorized
|Post View:
Css border or so-called border css these properties could allow us to determine the style and the border color of an element. Here I will give some examples of commonly used and may be useful.

SOME TYPES OF BORDER CSS
    1. none
    2. dotted
    3. dashed
    4. solid
    5. double
    6. groove
    7. ridge
    8. inset
    9. outset     		: Do not define the border line
    : Defines a dotted line border.

    Defining a full line border.
    Defining a double line border.
    : Defines a 3D grooved border line.
    : Defines a 3D ridged border line.
    : Defines a 3D inset border line.
    : Defines a 3D outset border line.
How to use it:
  1. none
    2. <p style = "border: 3px none; # 000000; padding: 20px;"> Do not define the border line </ p>
    Does not define the border line
  2. dotted
    3. <p style = "border: 3px dotted; # 000000; padding: 20px; align: center;"> Defines a dotted line border. </ p>
    Defining a dotted line border.
  3. dashed
    4. <p style = "border: 3px dashed; # 000000; padding: 20px;"> Define border dotted line. </ p>
    Defining a dotted line border.
  4. solid
    5. <p style = "border: 3px solid; # 000000; padding: 20px;"> Defines a full line border. </ p>
    Defining a full line border.
  5. double
    6. <p style = "border: 5px double; # 000000; padding: 20px;"> Defines a double line border. </ p>
    Defining a double line border.
  6. groove
    7. <p style = "border: 10px groove; # 000000; padding: 20px;"> Defines a 3D grooved border line. </ p>
    Defines a 3D grooved border line.
  7. ridge
    8. <p style = "border: 10px ridge; # 000000; padding: 20px;"> Defines a 3D ridged border line. </ p>
    Defines a 3D ridged border line.
  8. inset
    9. <p style = "border: 10px inset; # 000000; padding: 20px;"> Defines a 3D inset border line. </ p>
    Defines a 3D inset border line.
  9. outset
    10. <p style = "border: 10px outset; # FF3300; padding: 20px;"> Defines a 3D outset border line. </ p>
    Defines a 3D outset border line.
