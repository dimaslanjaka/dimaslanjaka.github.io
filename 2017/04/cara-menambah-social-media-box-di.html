2017-04-17

(Rating Section).<br><br>Contoh punya saya:<br><br><pre class="tr_bq"><amp-iframe width=300 height=122 sandbox='allow-scripts allow-same-origin allow-popups allow-popups-to-escape-sandbox' layout='responsive' frameborder='0' scrolling='no' src='https://source.l3n4r0x.cf/php/social.php?tw=DimasSkynetCybe&fb=secretnetworkforces'></amp-iframe></pre><br>Taruh kode tersebut sesuka hati anda. Dimanapun tempat nya pasti OK. Sesuaikan sendiri width=xxx nya. Height nya jangan karena itu menggunakan AMP-IFRAME kata lain IFRAME gak bisa se responsive memakai kode lainnya seperti AMP-FORM. Height tersebut sudah sesuai dengan widget height nya. Bila dipanjangkan maka BLANK putih menjalar dibawah Icon Icon itu.
