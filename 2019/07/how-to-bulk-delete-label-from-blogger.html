How to bulk delete category label from blogger posts

2019-07-21

Previous: How to bulk add category label to blogger posts

  1. Filter and Select category label you want to remove
  2. Select the post that will be removed from the label
  3. Select again the selected label from label selector
  4. Selected label from the selected posts have been deleted.

    If no posts at that label, the label will be removed automatically from your blog label lists
