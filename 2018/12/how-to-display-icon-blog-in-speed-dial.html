How to Display Icon Blog In speed dial Browser

Posted on 2018-12-25 In uncategorized
img src=https://res.cloudinary.com/dimaslanjaka/image/fetch/https://imgdb.net/images/4624.png title=speed dial icon alt=speed dial

Input Below Codes into <head>HERE</head>

<link rel='icon' sizes='192x192' href='URL_IMAGES' />
<link rel='shortcut icon' href='URL_IMAGES' sizes='192x182' type='image/x-icon'>
<link rel='apple-touch-icon' href='URL_IMAGES' />
<meta name='msapplication-square310x310logo' content='URL_IMAGES' />
<meta name='msapplication-TileImage' content='URL_IMAGES?w=270' />
