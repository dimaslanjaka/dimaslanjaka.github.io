Jika alamatnya <em>127.0.0.1 </em>, Server hanya mendengarkan koneksi lokal (di dalam perangkat atau telepon). <br>Dan port nya harus ada nomor lebih dari 1024. <em>rootdirectory </em> Adalah jalur dasar yang berfungsi sebagai direktori web root, dengan semua sumber termasuk JavaScript, gambar, stylesheet CSS, file HTML, dan skrip PHP. <br><h2> Haruskah saya menggunakan termux lighttpd, Apache, NGINX atau PHP built-in Web Server? </h2>Biasanya PHP digunakan di lingkungan produksi bersama dengan server Web lainnya seperti lighttpd, Apache atau NGINX karena server Web tersebut telah dioptimalkan untuk menangani muatan tinggi. <br>Namun pada ponsel Anda hanya akan memiliki pengguna Anda sendiri mengakses server Web.
