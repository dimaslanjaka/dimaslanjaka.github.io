How To Modifying Post Views On Homepage To Be Different

Posted on 2018-11-23 In uncategorized Disqus:
Sometimes we as the blog owner was bored to see the look of the blog, one of them post view on the homepage.

Sometimes we as the blog owner was bored to see the look of the blog, one of them post view on the homepage.&nbsp;Well, we can get rid of this boredom by modifying the post appearance on the homepage to be different.&nbsp;


We can freely modify the post homepage display is different, eg combine the grid and list view, or we modify the first post view is different from other posts.&nbsp;

This trick I have applied to some Company template.&nbsp;If you would like to try it, please follow the way below.&nbsp;

Please find the code below&nbsp;


<b:includable id='main' var='top'>

Scroll down a bit, then please delete the following code:&nbsp;
        <b:if cond='data:post.isDateStart'>
         <b:if cond='data:post.isFirstPost == &quot;false&quot;'>
            &lt;/div&gt;&lt;/div&gt;
          </b:if>
        </b:if>
        <b:if cond='data:post.isDateStart'>
          &lt;div class=&quot;date-outer&quot;&gt;
        </b:if>

        <b:if cond='data:post.isDateStart'>
          &lt;div class=&quot;date-posts&quot;&gt;
        </b:if>
Then scroll again down, and then delete the following code:&nbsp;

<b:if cond='data:numPosts != 0'> 
&lt;/div&gt;&lt;/div&gt; 
</b:if> 

Well now we just make a style to set the post view by using&nbsp;.post-outer:nth-child()&nbsp;

If we want to change the look of the first post, then all the styles associated with the first post use&nbsp;.post-outer:nth-child(1)&nbsp;as below for example:&nbsp;

.post-outer:nth-child(1) .post { 
.............. 
.............. 
} 
.post-outer:nth-child(1) .post h2 { 
.............. 
.............. 
} 
.post-outer:nth-child(1) .post-body { 
.............. 
.............. 
} 

Etc.&nbsp;

Then if you want to change the appearance of other posts also then add a style like this, eg for the second post, third, and fourth (in order to have the same look).&nbsp;

.post-outer:nth-child(2) .post, 
.post-outer:nth-child(3) .post, 
.post-outer:nth-child(4) .post { 
.............. 
.............. 
} 
.post-outer:nth-child(2) .post h2, 
.post-outer:nth-child(3) .post h2, 
.post-outer:nth-child(4) .post h2 { 
.............. 
.............. 
} 
.post-outer:nth-child(2) .post-body, 
.post-outer:nth-child(3) .post-body, 
.post-outer:nth-child(4) .post-body { 
.............. 
.............. 
} 

Etc.&nbsp;

And for other posts, create a style as usual, as follows&nbsp;

.post { 
.............. 
.............. 
} 
.post h2 { 
.............. 
.............. 
} 
.post-body { 
.............. 
.............. 
} 

How easy is not it?&nbsp;Good luck…&nbsp;
