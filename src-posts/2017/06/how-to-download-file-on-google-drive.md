For those of you who want to download files shared on <kbd>Google Drive</kbd> may have experienced access limits from shared files, be they data like Word or video. This is because visitors who access the file quite a lot at the same time, as a result we have to wait in accordance with the time specified from the Google Drive notification page.<br><br><img alt="GDrive" height="auto" src="https://res.cloudinary.com/dimaslanjaka/image/fetch/https://lh6.ggpht.com/k7Z4J1IIXXJnC2NRnFfJNlkn7kZge4Zx-Yv5uqYf4222tx74wXDzW24OvOxlcpw0KcQ=w300" title="GDrive" width="100%"><br><br>But do not worry, here I will try to share tips on how to open access that already limit without having to wait long.<br><br><h3>How to Download a File on Google Drive that Limit Access</h3>1. Open an inaccessible Google Drive file like the image below<br><br><figure><br><img alt="case" height="auto" src="https://res.cloudinary.com/dimaslanjaka/image/fetch/https://4.bp.blogspot.com/-YYXcOPH3g8Y/WTakMatwZMI/AAAAAAAAFK0/8PeWbd0nrAs-1dgwLIK2AwUyErejP41iACLcB/s1600/Cara%2BDownload%2BFile%2Bdi%2BGoogle%2BDrive%2Byang%2BLimit%2BAkses%2B1.png" title="case" width="100%"><br><figcaption>How to Download a File on Google Drive that Limit Access</figcaption><br></figure><br><br>2. Copy the <b>ID code</b> of the file into notepad (show screenshot below).<br><img height="auto" src="https://res.cloudinary.com/dimaslanjaka/image/fetch/https://1.bp.blogspot.com/-Xzbhr5DGob4/WTalzQ3Y6LI/AAAAAAAAFK4/66-Cn-QbIF4uiHmPZtfJzad83rb1ZthoACLcB/s1600/Cara%2BDownload%2BFile%2Bdi%2BGoogle%2BDrive%2Byang%2BLimit%2BAkses%2B2.png" width="100%"><br><br>3.
