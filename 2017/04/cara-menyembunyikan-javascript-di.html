Bagaimana cara menyembunyikan kode javascript di mobile blogger ?


Nah bagi yang memakai template AMP di blogger biasanya bingung bagaimana cara menaruh javascript di template AMP agar hanya di eksekusi pada desktop atau komputer saja.
Nah inilah fungsinya script conditional tag ini.
Berikut kodenya:

<b:if cond='data:blog.isMobileRequest == "false"'>
    JavaScript Yang Akan Disembunyikan
    </b:if>
kode diatas berfungsi menyembunyikan javascript di mobile request. Dan tujuannya agar AMP tidak error karena ada script tag di mobile request.

Taruh kode tersebut dimana saja, saya rekomendasikan di sebelum "</body>" tujuannya agar tidak memperlambat loading blog bila ditaruh di "<head>".
