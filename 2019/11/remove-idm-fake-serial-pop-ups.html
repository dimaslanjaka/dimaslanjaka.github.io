How to get rid of IDM Fake Serial Pop-ups

  1. Make sure the latest version of IDM is installed on your computer
  2. Turn off your antivirus because some antiviruses like MSE consider the Patch file to be a virus, when in fact it is not a virus, just a false alarm
  3. Download the Anti Fake Serial Patch through the link above
  4. Extract the file that you downloaded earlier with WinRAR (MANDATORY!)
  5. Check the system tray icon, if the IDM icon appears, right-click> Exit
  6. Open the Patch folder, then right-click Patch.exe> ​​Run as Administrator
  7. Click PATCH, then fill in your first name & last name> OK
  8. It's done!
