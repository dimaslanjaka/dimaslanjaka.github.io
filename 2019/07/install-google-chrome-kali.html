Install google chrome kali linux/debian/etc

2019-07-21

<div dir="ltr" style="text-align: left;" trbidi="on">open terminal, type:<br> 64 bit<br><pre>wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb<br></pre> 32 bit <br><pre>wget https://dl.google.com/linux/direct/google-chrome-stable_current_i386.deb<br></pre>wait until download complete, the type: <br> 64 bit<br><pre>dpkg -i google-chrome-stable_current_amd64.deb<br></pre> 32 bit <br><pre>dpkg -i google-chrome-stable_current_i386.deb<br></pre> then, fix chrome installation and depencies: <pre><br>apt update -y && apt upgrade -y && apt --fix-broken install<br></pre> done, find chrome in app lists, now available. good luck </div><img src="https://res.cloudinary.com/dimaslanjaka/image/fetch/https://www.pngarts.com/files/3/Linux-PNG-Image-Background.png">
