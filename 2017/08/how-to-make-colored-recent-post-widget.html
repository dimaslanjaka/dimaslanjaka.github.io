How to Make Recent Post Widget Color

Go to Blogger > Themes > Click the Edit HTML button
Include material icon before </head> like this:
<link href=‘https://fonts.googleapis.com/icon?family=Material+Icons’ rel=‘stylesheet’ type=‘text/css’/>
