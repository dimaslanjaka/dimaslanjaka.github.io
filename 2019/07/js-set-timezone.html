[JS] Set Timezone

2019-07-17
Set timezone javascript without using any plugins

const DATE = new Date().toLocaleString('en-US', {
  timeZone: 'Asia/Jakarta'
});
console.log(DATE); //debug
document.write(JSON.stringify(DATE)); //debug dynamically print
