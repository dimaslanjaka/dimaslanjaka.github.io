</h2>Untuk mentransfer skrip PHP dan file lainnya dengan mudah di antara workstation dan perangkat Android, server FTP adalah suatu keharusan. <br>Sangat mudah untuk mengatur lingkungan desktop Anda untuk menggunakan server FTP LAN. Anda bisa memasang FTP client di sistem, atau bahkan menggunakan Mozilla Firefox (ekstensi FireFTP). Di Windows, ketik Explorer Address Bar, URL server, seperti: <br>ftp://user@ipaddress:port/  <br>Hal yang sama bisa dilakukan untuk lingkungan desktop sistem operasi lain, seperti Nautilus for Linux, dll.  <br><h2> Menjalankan PHP dari Termux dalam endless loop  </h2>Untuk polling endlessly beberapa kondisi. Di dalam naskah, loop harus memiliki sebuah    <em>sleep(detik)   </em> call, untuk menghindari memonopoli prosesor.  <br>Pada konsol Termux, ketik:  <br><pre>php endlessloop.php<br></pre><h2> Kontrol Tablet Android dengan Telepon via Web Server </h2>Salah satu cara untuk mengendalikan tablet atau jenis perangkat seluler lainnya dari ponsel adalah dengan menjalankan server Web pada perangkat tablet dan mengaksesnya melalui browser yang berjalan di telepon. <br>Jika Anda menjalankan PHP di perangkat tablet sebagai server Web dengan Termux, akan sangat mudah memprogram aplikasi sederhana yang bisa melakukan apapun yang Anda inginkan untuk mengendalikan tablet. <br><h2> Berjalan sebagai Android Server untuk Aplikasi Web Lokal PHP </h2><div>Anda dapat menjalankan PHP dan membuatnya bekerja sebagai server Web yang menjalankan skrip PHP Anda di sisi direktori akar proyek Anda yang tersimpan dalam sistem file memori perangkat Anda. </div>Di Termux, ketik Untuk memulai the development <b>Web server embedded</b> in PHP.  <br><code> php -S address:port -t /sdcard/php/rootdirectory/    <br>  </code>  <br>Jika Anda menggunakan alamat yang ada   <em>0.0.0.0   </em>, dan Server akan mendengarkan salah satu alamat IP jaringan perangkat. Jika alamatnya    <em>127.0.0.1   </em>, Server hanya mendengarkan koneksi lokal (di dalam perangkat atau telepon).  <br>Dan port nya harus ada nomor lebih dari 1024.    <em>rootdirectory   </em> Adalah jalur dasar yang berfungsi sebagai direktori web root, dengan semua sumber termasuk JavaScript, gambar, stylesheet CSS, file HTML, dan skrip PHP. <br><h2> Haruskah saya menggunakan termux lighttpd, Apache, NGINX atau PHP built-in Web Server? </h2>Biasanya PHP digunakan di lingkungan produksi bersama dengan server Web lainnya seperti lighttpd, Apache atau NGINX karena server Web tersebut telah dioptimalkan untuk menangani muatan tinggi. <br>Namun pada ponsel Anda hanya akan memiliki pengguna Anda sendiri mengakses server Web. Oleh karena itu, server Web PHP built-in cukup baik untuk tujuan ini, kecuali Anda memerlukan beberapa fitur server Web tertentu yang disediakan oleh server Web berkinerja tinggi. <br><h2> Menggunakan class PHP Termux API </h2><pre>    $termux = new termuxapi();<br>echo $termux-&gt;location();<br></pre><h2> Masalah Termux:API  </h2>Beberapa Fungsi Termux:API tahan lama dan padat sumber daya seperti misalnya cek kotak masuk SMS, daftar kontak, dan lokasi. <br>Karena program Termux disebut dengan   <em>shell_exec()   </em>, Perilaku sinkronnya dapat menunda dengan serius pendekatan Web Server. Pertimbangkan untuk membuat panggilan AJAX ke metode untuk menyembunyikan masalah ini. <br><h2> Solusi Masalah  </h2>Termux adalah aplikasi Android yang bagus yang memudahkan menjalankan PHP atau program lain yang biasanya tersedia di distro Linux. <br>Kemungkinan untuk menjalankan PHP baik sebagai script baris perintah atau sebagai server Web membuka segudang kemungkinan untuk menjalankan aplikasi yang bagus dengan menggunakan kode PHP seperti aplikasi Android asli lainnya. <br>Jika Anda menyukai artikel ini, bagikan dengan teman pengembang Anda.
